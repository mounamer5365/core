## ✅ First-time setup

When core opens for the first time, you can set up the parts you need.

### Add a server
1. Open the Servers area.
2. Choose Add server.
3. Enter the server name, host, username, and password or SSH key.
4. Save the connection.
5. Test it to make sure it works.

### Add a database
1. Open the Databases area.
2. Choose Add database.
3. Enter the server address and login details.
4. Save the entry.
5. Open it to view or manage data.

### Add a container
1. Open the Containers area.
2. Connect to a Docker host.
3. Select the container you want.
4. Start, stop, or inspect it.

### Set up a tunnel
1. Open the Tunnels area.
2. Add the target host and port.
3. Save the tunnel.
4. Turn it on when you need remote access.

## ❓ Troubleshooting

### core does not open
- check that the download finished
- make sure you extracted the .zip file if needed
- try running the app as an administrator
- restart Windows and open it again

### The app says it cannot connect
- check your internet connection
- confirm the server address is correct
- confirm your username and password are correct
- check that SSH or database access is allowed on the target system

### Windows blocks the app
- right-click the file and choose Properties
- if you see an Unblock box, enable it
- try opening the app again
- if the file came from the Releases page, download it again

### A workflow fails
- open the workflow and check each step
- make sure the target server is online
- confirm the container or database still exists
- run the steps one by one to find the issue
